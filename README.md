# Easy-ML

Easy-ML is a machine learning framework makes machine learning easier based on sk-learn\keras.
All models that are integrated include：

* Nearest Neighbors
* Bagging
* GradientBoosting
* SGD
* LibSVM
* Linear SVM
* SMO
* LinearSVC
* Decision Tree
* Random Forest
* AdaBoost
* Naive Bayes
* Neural Network
* ......

python main.py
